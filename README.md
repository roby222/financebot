# FinancePalmirioBot

Dashboard ETF per macro-trend geopolitici (Iran-USA, Russia-Ucraina).

## Avvio

```bash
python3 -m http.server 8080 --directory static
```

Poi apri `http://localhost:8080` nel browser.
In alternativa apri direttamente `static/index.html`.

## Documentazione

La guida completa è integrata nel sito — clicca **?** in alto a destra.
